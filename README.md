<h1>Sistema cadastro de cliente</h1>

Sistema feito para cadastrar um cliente atraves do Swagger com os seguintes campos:

Nome, Sobrenome, Rua, Bairro, Numero

Tecnologias ultilizadas:

C# <br>
.Net <br>
Entity Framework Core <br> 
Sql Server <br>
